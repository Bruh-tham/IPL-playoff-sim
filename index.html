NRR Mode
NRR Mode Guide

Predicting exact scores allows the simulator to calculate dynamically shifting Net Run Rates for more accurate probabilities!

  • 1. Select Winner: Click a team's logo to tentatively select them (or 'No Result').
  • 2. Input Scores: Type predicted Runs and Overs.
  • 3. Submit: Click Add to run the simulator and lock in the math.

Note: Overs use standard cricket math (e.g. 19.4 = 19 overs, 4 balls). If a team gets bowled out, technically log their overs as 20.0.
